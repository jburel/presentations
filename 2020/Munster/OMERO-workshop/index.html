Prepare the training: 1

  • Set a date
  • Book a room
  • Test internet connection inside the room
  • Check that OMERO.insight imports an image from the training room
  • Think about what setup you need, whether you ask users to bring their laptop

Prepare the training: 2

  • Plan the content of the workshop: OMERO guide walkthroughs
  • Plan the content of the workshop: see past workshops
  • Plan the installation of OMERO.insight if you want to show imports
  • Plan the OMERO extensions and OMERO add-ons you need to install on the training machines and on the server
  • Plan the 3rd party software you need to install on the training machines
